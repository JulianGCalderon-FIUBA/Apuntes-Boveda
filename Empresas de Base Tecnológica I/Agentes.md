Existen tres tipos de agentes.

- Banco central
- Bancos comerciales
- Personas

## Balance

El balance de un agente es el resumen de sus [[Activos]] y sus [[Pasivos]].

### Banco Central

| ACTIVOS        | PASIVOS    |
| -------------- | ---------- |
| Oro y Divisas  | Circulante |
| Otros Activos  | Reservas   |
| Prest. al Gob. | Bonos BCRA |

### Bancos Comerciales

| ACTIVOS            | PASIVOS   |
| ------------------ | --------- |
| Préstamos          | Depósitos |
| Reservas           |           |

### Personas

| ACTIVOS    | PASIVOS |
| ---------- | ------- |
| Circulante | Riqueza |
| Depósitos  |         |
